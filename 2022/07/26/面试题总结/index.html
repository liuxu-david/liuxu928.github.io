面试题总结

面试题总结

数据类型

基本类型：Numeber，String，Boolean，undefined，null，symbol，bigint

引用类型 Object（普通对象，Array，Date,正则，Math 数学对象），function 类型

map 和 foreach

foreach 和 map 两者都可以对数组每一项进行遍历，map 会对每一个 item 做操作然后返回一个新的数组，而 foreach 不会返回，只会更改原始数组的数据

for（…in …）可以遍历得到对象的 key 值；

检查字符串

indexof（）检查字符串或者数组

js 中的变量提升

js 是单线程语言，执行的时候是按照顺序来执行的，但不是逐行分析和执行的，而是一段一段的分析执行，在执行前，会检测所有变量和函数声明，把他们添加到内存中去，所以这些变量和函数 声明能在真正声明之前使用。

作用域

作用域就是一个独立的地盘，不会让变量暴露出去，最大的作用就是隔离变量，不同作用域下同名的变量不会有冲突

ES6 之前没有块级作用域，只有全局作用域和函数作用域。

arguments

在调用有参函数的时候，就会把所有参数都存储到 arguemnts 的对象中，是一个类数组数据

作用：可以先动态设置形参，然后通过 arguemnts 动态的的加入参数

var 和 let 和 const 区别

变量提升：var 可以变量提升，let 和 const 没有变量提升

初始化值：var 和 let 可以不设置初始化值，但是 const 必须给初始化值

重复声明：var 可以重复声明，后面声明的同名变量会覆盖前面声明的变量；

块级作用域：var 不具有块级作用域，let 和 const 具有块级作用域。

指针指向：const 不会改变指针指向，但是 var 和 const 会改变指针指向；

display 有哪些属性

1、none，inline，inline-block，block，flex，table，

块元素和行内块元素和行内元素区别

块元素：一行只能放一个块级元素，可以设置宽度和高度，

行内块元素：一行可以放好几个行内块元素，可以设置高度和宽度

行内元素：一行可以放多个行内元素，不可以直接设置宽度和高度；

什么是虚拟 Dom

如果浏览器操作真实 Dom 就需要付出昂贵的代价，而且频繁的操作真是 Dom 容易引发问题，所以就出现了虚拟 Dom，虚拟 Dom 作用就是每一次响应式数据发生改变引起页面重新渲染的时候，虚拟 Dom 进行比较，然后更新真实 Dom，尽可能的减少更新的真实 Dom，从而达到优化性能的目的

回流和重绘

第一次确定节点的位置或者大小，称之为布局，之后修改了节点的大小或者位置，引发了页面重新计算称之为回流。

引起回流？

1，DOM 结构发生改变（添加节点或者移除节点）

2，页面的布局发生改变（width，height，padding，）

3，窗口发生改变

第一次渲染内容称之为绘制，之后页面重新渲染称之为重绘

引起回流？

1，修改背景颜色

2，修改边框的样式

3，修改文字颜色

怎样能避免回流和重绘呢？

1，修改样式的时候尽量一次性修改，添加 class

2，尽量避免频繁操作 DOM 元素

3，对某些元素使用 position 的 fixed 或者 absolute

闭包

定义：一个函数可以访问到外层作用域的自由变量，那么这个函数和周围的环境就是闭包

从广义的角度，js 中的函数都是闭包，

从狭义的角度来说，js 中的一个函数可以访问到外部的自由变量，那么就是闭包。

splice 和 slice 的区别？

splice 会改变原始数组，而 slice 不会改变原数组 ，splice 可以删除数组元素也能插入元素，

splice 有 3 个参数，第一个参数表示开始修改元素的位置，第二个表示需要移除数组元素的个数，第三个参数填写需要添加的元素

slice 有 2 个参数，开始提取数组元素的索引，结束提取元素的索引

vue 中 v-if 和 v-show 区别

v-if 是动态向 dom 树中添加或者删除 dom 元素，而 v-show 则是通过控制 dom 元素的 display 属性来控制显隐的

v-if 切换消耗较大性能，v-show 首次渲染有较大消耗

如果需要频繁切换显示隐藏使用 v-show’比较好，当不大可能改变的时候使用 v-if
