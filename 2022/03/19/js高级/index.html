Object: 任意对象

Array： 一种特殊数对象，（有数值下标属性，内部数据有序）

instanceof

判断案例

问题

undefined 和 null 的区别？

undefined 是代表定义了但是没有赋值，null 是定义并且赋值了但是赋值为 null

什么时候给变量赋值为 null?

让赋值的变量成为垃圾对象被垃圾回收器回收，出事赋值表示要赋值为对象

数据变量内存

什么是数据？

存储在内存中代表特定信息的东西

特点：可运算，可传递

什么是变量

可变化的数据，由变量名和变量值组成，每个变量都对应一块小内存，变量名用来查找对应的内存，变量值就是在内存中保护的

什么是内存？

就是内存条通电以后产生的可以存储数据的空间

三者关系

内存是用来存储数据的空间，变量是内存的标识

var a=xxx,a 内存中保存的到底是什么？

xxx 为基本类型的时候，保存的就是这个数据

xxx 为对象时，保存的就是对象的地址值

xxx 为变量时，保存的可能是基本数值，也可能是对象的地址值

引用赋值问题**

2 个引用变量指向同一个对象的时候，，一个变量修改对象的内部,另一个引用变量都能看到修改以后的数据

2 个引用变量指向同一个对象的时候，让其中的引用变量指向另一个对象，另一个引用变量依然指向前一个对象

js 传递变量参数时，是值传递还是引用传递？

可能是值传递，也有可能是引用传递（地址值）也可以理解为都是值传递（基本值传递，地址值传递）

js 引擎如何进行内存管理

释放内存：

局部变量：函数执行完以后自动释放；

对象：先成为垃圾对象==》然后被垃圾回收器回收

对象

什么是对象？为什么用

拥有属性和方法的数据，用来保存多个数据的容器

统一管理多个数据

方法是特殊的属性==》属性值是函数

对象使用中什么时候必须使用[“属性名”]？

1 属性名中包含特殊字符：- 空格

2 属性名不确定

函数

是什么？可以执行，实现一定功能的封装体

为什么？提高代码的复用，便于阅读

怎么封装？函数声明，表达式

什么函数是回调函数？

1 你定义的

2 你没有调

3 最终它执行了

常见回调函数===>dom 事件回调函数，定时器

IIFE 函数

匿名自调用函数（立即调用函数表达式）

作用：隐藏实现，不会污染全局命名空间，用它来编写 js 模块

函数中的 this

如果没有指定就是指 window

所有函数内部都有一个变量 this

它的值是调用函数的当前对象

如何确定 this 的值？

test()指 window

p.test()指 p

new test（）指新建的对象

p.call(obj)：obj
